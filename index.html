Anatomie du cœur (niveau débutant)

1. Rôle du cœur

Le cœur est une pompe musculaire qui fait circuler le sang dans tout l’organisme. Il envoie le sang vers les poumons pour l’oxygéner et vers les organes pour leur apporter oxygène et nutriments.

2. Les quatre cavités

Le cœur est divisé en quatre cavités :

  • Oreillette droite : reçoit le sang pauvre en oxygène provenant du corps.
  • Ventricule droit : envoie ce sang vers les poumons (circulation pulmonaire).
  • Oreillette gauche : reçoit le sang riche en oxygène venant des poumons.
  • Ventricule gauche : envoie ce sang vers tout le corps (circulation systémique).

3. Les principales valvules

Les valvules cardiaques assurent un sens unique à l’écoulement du sang :

  • Valvule tricuspide : entre oreillette droite et ventricule droit.
  • Valvule pulmonaire : entre ventricule droit et artère pulmonaire.
  • Valvule mitrale (biscuspide) : entre oreillette gauche et ventricule gauche.
  • Valvule aortique : entre ventricule gauche et aorte.

4. Les gros vaisseaux à connaître

  • Veines caves supérieure et inférieure : amènent le sang pauvre en O₂ au cœur droit.
  • Artère pulmonaire : envoie le sang du cœur droit vers les poumons.
  • Veines pulmonaires : ramènent le sang oxygéné des poumons au cœur gauche.
  • Aorte : distribue le sang oxygéné à tout l’organisme.

5. À retenir pour les examens

  • Le cœur a 4 cavités : 2 oreillettes, 2 ventricules.
  • Le côté droit gère la circulation pulmonaire (vers les poumons).
  • Le côté gauche gère la circulation systémique (vers le reste du corps).
  • Les valvules empêchent le reflux du sang (circulation à sens unique).

Contact / Installation sur votre domaine

Remplacez ce bloc par votre formulaire de contact ou vos coordonnées.

Étape 1
Choisir un nom de domaine

Exemple de domaine : human3dclinic.com (ou un nom de votre choix). Vous pouvez l’acheter sur un registrar (OVH, Namecheap, LWS, etc.).

Étape 2
Hébergement web

Prenez un hébergement simple (mutualisé) qui permet de déposer des fichiers HTML. Vous aurez un accès FTP ou un gestionnaire de fichiers.

Étape 3
Envoyer la page

Uploadez ce fichier index.html (et le dossier models avec vos modèles 3D) dans le dossier www, public_html ou similaire de votre hébergeur.

Étape 4
Associer domaine & site

Dans le panneau de votre hébergeur, faites pointer le domaine vers le dossier où vous avez mis index.html. Attendez la propagation DNS (quelques heures), puis testez dans le navigateur.
